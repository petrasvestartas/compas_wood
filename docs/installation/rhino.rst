********************************************************************************
Rhino
********************************************************************************

.. code-block:: bash

    open Rhino8
    in the Rhino command-line type: PackageManager
    in the PackageManger search for a compas_wood package
    click button install
    restart Rhino8
    open Grasshopper
    drag and drop install component
    try examples from the food4Rhino page
